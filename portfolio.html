PROJECT I WORKED ON..

TEXT TO SPEECH CONVERTOR

The Text to Speech Converter project allows users to enter text into a textarea provided in the user interface. When the "Convert to Speech" button is clicked, the project utilizes the Web Speech API to convert the entered text into speech. The converted speech is then played back to the user. The user interface is designed using HTML and CSS, with a clean and intuitive layout. The container element holds the header, textarea, and convert button. The header displays the title of the project, which is "Text to Speech Converter." The textarea allows users to input the text they want to convert, and the convert button triggers the conversion process.
